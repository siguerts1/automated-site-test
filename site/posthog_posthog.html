posthog-cli-v0.0.4 (2025-03-21)

## Install posthog-cli 0.0.4

### Install prebuilt binaries via shell script

```sh
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/PostHog/posthog/releases/download/posthog-cli-v0.0.4/posthog-cli-installer.sh | sh
```

### Install prebuilt binaries via powershell script

```sh
powershell -ExecutionPolicy Bypass -c "irm https://github.com/PostHog/posthog/releases/download/posthog-cli-v0.0.4/posthog-cli-installer.ps1 | iex"
```
